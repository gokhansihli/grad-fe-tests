# Front-end Tech Tests

This repository contains a collection of technical tests I completed following my graduation from **Northcoders**.

## Tech Stack

- **Frontend:** React.js
- **Styling:** CSS3
- **Logic:** JavaScript (ES6+), Hooks
- **State Management:** `useState`, `useEffect`

## Tests

### 1. **Login Page**

- Built a **Login and Signup** system
- Implemented validation logic for password
- Handled error messaging

### 2. **Movie Search**

- Created a filterable movie database that searches through titles, directors, genres, and cast
- Integrated a `setTimeout` within `useEffect` to prevent unnecessary re-renders
- **Data Transformation:** Logic to calculate average ratings from multiple sources.

### 3. **Region Guessing Game**

- A 60-second challenge to name all 50 regions
- Uses **Regex matching** to handle case-insensitive user input and prevents duplicate correct entries
- Features a "Give Up" function that reveals missing answers

### 4. **Find X Game**

- A coordinate-based grid game where the target "X" moves randomly after every find
- Implemented a timer to track "Best Time" and a score counter to track total finds

### 5. **Dropdown Menu**

- A multi-level dropdown menu built using CSS and React component structure

### 6. **Chat App**

- A two-way chat interface where multiple users can post to a shared "General Chat" box
- Focused on managing shared state

## Installation & Setup

1. **Clone the repo:**
   ```bash
   git clone https://github.com/gokhansihli/grad-fe-tests.git
   ```
2. **Install:**
   ```Bash
   npm install
   ```
3. **Start:**
   ```Bash
   npm start
   ```
